Classes
 ClassDescription
AppConfigFactory Configuration factory that understands app.config file format. It is focused only on specific section of the configuration file (i.e. section reserved for SDK) and is able to detect changes of the configuration in run-time.
AppConfigWrapper Problem is that users cache the ISdkConfiguration instance they get, and app.config is a bit awkward as a new instance has to be obtained all the time using GetSection(). This is a wrapper so that the underlying section can change, but the client has just this instance cached.
